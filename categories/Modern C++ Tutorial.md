# Modern C++ Tutorial

Modern C++ 是和C++98截然不同的存在。实际上Modern C++这个概念本身也在更新换代，以C++11~C++17为代表的Modern C++应该说是现在的主流，而这里的Modern其实还在Future。

本tutorial的目标受众并不是对C++以及程序设计一无所知的小白。我希望读者至少知道一般C++的语法，例如能够实现一个简单的贪吃蛇小游戏之类。
